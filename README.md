# Weather Dashboard

An interactive and responsive Weather Dashboard built using JavaScript (ES6+), Fetch API, and async/await. The application supports geolocation-based weather detection, city search functionality, and a dynamic 5-day forecast display. Designed with a clean UI using CSS Grid and modular JavaScript architecture.
